Improvised Bubble Sort

Improvised Bubble Sort Algorithm is just a way of improving the Time Complexity in the best case of already sorted.
This makes the time complexity O(n) in case of best-case instead of O(n2).While the complexity of average and worst case will remain same as of bubble sort i.e. O(n2).

Algorithm

Improvised_Bubble_sort(A, n)
1. Pass = 1
2. For i = 0 to (n -1 && pass==1)
3. Pass= -1
4. For j = i+1 to n
5. If A[i] > A[j]
6. Then swap A[i] & A[j] && Pass = 1
7. End if
8. End for
9. End for

{
int i,j,temp,min;

for(i=0;i< n-1 && flag==1;i++)
{
flag=-1;
count++;
for(j=i+1;j< n;j++)
{
count++;
if(a[i] > a[j])
{
count++;
temp=a[i];
count++;
a[i]=a[j];

}

}
